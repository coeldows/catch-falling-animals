
Catch the Falling Animals - Simple Web Game

How to play:
- Use Arrow keys to move left, right, up, and down.
- Catch the falling animals with your catcher.
- Score points and gain hearts every 5 catches in a row.
- Pause, Restart, and toggle background Music!

Deployment:
- Open the index.html file locally, OR
- Upload it to GitHub Pages for free hosting!
